# 🌳 Binary Tree DSA (C++)

## 📖 Description
This repository contains implementation of Binary Tree in C++ including traversal methods and basic operations.

## 🚀 Features
- Inorder Traversal
- Preorder Traversal
- Postorder Traversal
- Level Order Traversal
- Height of Tree
- Count Total Nodes

## 🛠️ Technologies Used
- C++

## 📂 How to Run
1. Clone the repository
2. Open in VS Code or any C++ IDE
3. Compile:
   g++ main.cpp -o tree
4. Run:
   ./tree
